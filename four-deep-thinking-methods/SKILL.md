# Four Deep Thinking Methods

## Overview

Four questioning modes to guide deep thinking. Each mode uses one question at a time to progressively deepen understanding.

## Four Modes

### 1. Goal Decomposition (目标拆解法)
"我的目标是______，请以顶级策略顾问身份，每次只提一个问题，帮我将目标拆解为具体、可衡量、可执行的步骤，从宏观层面开始直至形成清晰计划。你的第一个问题是什么？"

### 2. Socratic Questioning (苏格拉底式提问)
"请对我使用苏格拉底式提问，我的想法是______，不要给答案或建议，仅通过环环相扣的问题引导我思考该想法的底层逻辑、潜在假设和边界条件。请提出第一个问题。"

### 3. First Principles (第一性原理思考)
"让我们用第一性原理思考问题：______，请像马斯克一样提问，帮我剥离行业惯例和假设，直抵问题最基础不可削减的事实。你的第一个问题是什么？"

### 4. Pre-Mortem (复盘前置)
"我们做复盘前置练习：假设项目'______'已启动且一年后彻底失败，请以调查员身份，每次只问一个问题，从最可能失败原因开始分析。你的第一个问题是什么？"

## Usage Example

**User input**: "用目标拆解法，我的目标是三个月内学会Python"
**AI action**: Acts as strategy consultant, asks first question to decompose goal from macro to actionable steps
**Expected result**: Single focused question to begin decomposition; continues one question at a time
