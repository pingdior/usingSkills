# How to Write Excellent Prompts

## Overview

Two fundamental principles for effective LLM prompt writing.

## Core Principles

### Principle 1
Everything that can be described in characters can be solved by LLM.

### Principle 2
When writing prompts, the most critical factors are: **goal clarity**, **clarity**, and **feasibility** - especially in constraint flow and workflow design.

## Key Focus Areas

- **Goal**: Explicit, clear, achievable
- **Constraints**: Well-defined flow
- **Workflow**: Logical, executable steps

## Usage Example

**User input**: "帮我写一个提取邮件地址的提示词"
**AI action**: Defines clear goal (extract emails), sets constraints (input format, output format), designs workflow (parse → extract → validate)
**Expected result**: Prompt with explicit goal, constraints, and workflow
